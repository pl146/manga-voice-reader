How it works

Four steps

Install, open manga, hit play. That's it.

01

Install the extension

Load into Chrome. The local server auto-starts on your Mac via launchd.

02

Open any manga

MangaFire, MangaDex, Webtoons, any site. MangaVoice works everywhere.

03

Hit play

One click. Page captured, bubbles detected, text extracted, voice generated.

04

Sit back

Auto-skip advances pages. Bubbles highlight as they're read. Full auto-pilot.
